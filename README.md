# Portfolio 2026

Personal portfolio of LicJoseAngelGodinez.

## Tech Stack

- React 19
- TypeScript
- Vite
- React Router

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for deployment on [Vercel](https://vercel.com).
